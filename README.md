# Dynamic Portfolio

A modern, interactive portfolio website designed to showcase my skills, projects, and professional journey as a Full Stack Developer.

Built using React, GSAP, and Tailwind CSS, the portfolio delivers a smooth and engaging user experience through dynamic animations, expandable content sections, responsive layouts, and interactive UI elements.

## Preview

The portfolio includes:

- About Me section
- Skills Showcase
- Project Gallery
- Contact Information
- Social Media Links
- Resume Download Section
- Interactive Expandable Cards
- Infinite Scrolling Carousel
- Smooth GSAP Animations

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/dynamic-portfolio.git
```

### Navigate to Project

```bash
cd dynamic-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Application runs at:

```text
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
